# Corona Mask Finder Flutter

## 플러터로 제작 코로나 마스크 파인더한(미완성)
1. Google Map
* 플러터 화면에 구글 맵을 구현하기 위해서 사용 Google Cloud Platform 아이디가 필요하다.

2. Geolocator
* 플러터 앱에 위치기반 정보를 가져오기위해 사용하였다.


3. Flutter Spinkit

* 로딩화면을 이쁘게 구현해주는 라이브러리다.

4. Http
* 웹에서 데이터를 파싱해주는 도구이다.
